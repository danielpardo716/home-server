## Services
- **NextCloud:** Network drive
- **Immich:** Photo/video library
- **Plex:** personal music library
- **Home Assistant**: Home automation

## Tools
- **NGINX Proxy Manager:** reverse proxy (host all services on the same domain)
- **PiHole:** Network-wide ad-blocker
- **Homarr:** Monitoring dashboard
- **Grafana:** Visualization
